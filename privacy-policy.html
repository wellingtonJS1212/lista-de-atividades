Politica de Privacidade

Aplicativo: Lista de Atividades

Ultima atualizacao: 18 de marco de 2026

Resumo

O aplicativo Lista de Atividades foi desenvolvido para ajudar o usuario a organizar tarefas com lembretes visuais e contagem regressiva. O app funciona localmente no dispositivo e nao foi projetado para coletar ou transmitir dados pessoais para servidores externos.

Dados coletados

Atualmente, o aplicativo nao coleta dados pessoais identificaveis, nao exige cadastro, nao solicita login e nao envia informacoes para servidores externos.

Armazenamento local

As tarefas criadas e configuracoes relacionadas aos lembretes podem ser armazenadas apenas localmente no dispositivo do usuario para permitir o funcionamento normal do app.

Compartilhamento de dados

O aplicativo nao compartilha dados com terceiros.

Permissoes sensiveis

O aplicativo nao utiliza localizacao, camera, microfone, contatos ou outras permissoes sensiveis para o funcionamento principal.

Seguranca

Como os dados permanecem localmente no dispositivo, o usuario mantem controle direto sobre as informacoes registradas no app.

Alteracoes nesta politica

Esta politica pode ser atualizada futuramente caso novas funcionalidades sejam adicionadas. Quando isso acontecer, a data de atualizacao desta pagina sera revisada.
