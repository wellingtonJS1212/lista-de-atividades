# Lista de Atividades 1.0.0

Primeira versao publica do aplicativo Lista de Atividades para Android.

## Destaques

- Criacao de tarefas com interface simples
- Lembretes visuais com tempo configuravel
- Contagem regressiva em tempo real
- Destaque visual para tarefas proximas do fim
- Distribuicao em APK fora da Play Store

## Arquivos

- `app-release-signed.apk`: instalacao direta no Android
- `app-release.aab`: pacote de distribuicao e backup de release

## Instalacao

1. Baixe o `app-release-signed.apk`
2. Abra o arquivo no Android
3. Autorize a instalacao de apps externos se necessario
4. Finalize a instalacao
