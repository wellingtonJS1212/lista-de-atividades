# Lista de Atividades 1.0.5

Nova atualização do aplicativo Lista de Atividades para Android.

## Destaques

- PDF salvo direto na pasta Downloads do celular (acessível pelo gerenciador de arquivos)
- Confirmação visual após salvar o PDF com botão para abrir a pasta Downloads
- Animação suave na troca entre as abas Tarefas e Histórico
- Botões Sair e Fechar app redesenhados como ícones redondos na barra superior
- Filtros 7, 15 e 30 dias do gráfico posicionados lado a lado no mobile

## Arquivos

- `app-release-signed.apk`: instalação direta no Android
- `app-release.aab`: pacote de distribuição e backup de release

## Instalação

1. Baixe o `app-release-signed.apk`
2. Abra o arquivo no Android
3. Autorize a instalação de apps externos se necessário
4. Finalize a instalação
